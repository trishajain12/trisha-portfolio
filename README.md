# Personal Portfolio Website

This is my personal portfolio website built while preparing for my Front-End React internship at Quicken. The site highlights my background, technical skills, work experience, projects, and contact information in a clean, responsive layout.

## About

I created this portfolio to strengthen my React, JavaScript, and CSS skills while building something practical for recruiters and professional networking. The project gave me hands-on practice with React components, reusable section layouts, responsive styling, project cards, navigation links, and media previews.

## Built With

* React
* JavaScript
* HTML/CSS
* Vite
* Git/GitHub

## Features

* Responsive personal landing page
* About Me section
* Work Experience section
* Projects section with GitHub links and demo preview
* Contact section
* Custom styling with CSS
* Organized React component structure

## Purpose

This project was built as part of my preparation for a Front-End Development internship at Quicken. My goal was to practice building clean, user-focused interfaces while becoming more comfortable with React, component-based development, CSS layouts, and GitHub project organization.
